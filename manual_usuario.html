Introducción

Pet-Health Services es una plataforma diseñada para gestionar todos los aspectos relacionados con la salud y bienestar de tus mascotas. Con nuestra aplicación, puedes acceder a servicios veterinarios, programar citas, realizar pedidos de productos y mucho más.

Diseño Visual

Nuestra aplicación utiliza una paleta de colores cuidadosamente seleccionada para una experiencia visual agradable:

  • Fondo principal: Beige claro para una experiencia relajante
  • Texto: Verde oliva para lectura cómoda
  • Elementos destacados: Verde oscuro para botones y encabezados
  • Superficies: Blanco para tarjetas y contenedores

Preguntas Frecuentes

¿Puedo tener múltiples mascotas en una misma cuenta? +

Sí, puedes registrar y gestionar todas tus mascotas desde una sola cuenta. No hay límite en el número de mascotas que puedes agregar.

¿Qué métodos de pago aceptan? +

Aceptamos tarjetas de crédito/débito, transferencias bancarias y pago en efectivo en clínica. También contamos con planes de financiamiento para procedimientos costosos.

¿Puedo cancelar o reprogramar una cita? +

Sí, puedes cancelar o reprogramar con al menos 24 horas de anticipación sin costo alguno. Las cancelaciones con menos tiempo pueden estar sujetas a cargos según la política de la clínica.

¿Cómo contacto a mi veterinario? +

Puedes enviar mensajes directos a través de la plataforma para consultas no urgentes. Para emergencias, utiliza el botón de "Urgencias" que conecta directamente con el servicio de emergencias 24/7.
